# Pedidos

Aplicación desarrollada en Java que gestiona pedidos de clientes. Este proyecto sirve como práctica de programación orientada a objetos y organización de proyectos con Gradle/Ant.

## Estructura del proyecto

```
pedidos/
├── build.xml           # Configuración de compilación (Ant)
├── manifest.mf         # Archivo de manifiesto
├── nbproject/          # Configuración del IDE (NetBeans)
├── src/                # Código fuente principal
│   └── ...             # Clases Java relacionadas con pedidos
└── README.md           # Documentación del proyecto

```

## Tecnologías utilizadas

* Lenguaje: Java

* Gestor de construcción: Ant / Gradle
